## What it changes/removes

- Add more Youtube videos per row
- Shorts shelves on the home page
- Category chips at the top (All, Gaming, Music, etc.)
- Notification bell and microphone button
- Sidebar links: Shorts, Explore, Music, Films, Gaming, News, Podcasts, and more
- "More from YouTube" sidebar section (Premium, Studio, Kids)
- YouTube Premium and promo pop-ups
- Recommended videos panel on watch pages. Just the video and comments.

## Settings

Every feature can be turned on or off individually. To open the settings page:

1. Right-click the BareTube icon in your browser toolbar
2. Select **Options**

Or go to `chrome://extensions`, find BareTube, click **Details**, then **Extension options**.

Changes take effect the next time you load a YouTube page.

> **Note:** The "Hide pop-ups and promo banners" option is on by default. Because YouTube uses the same pop-up system for everything, turning this on will also block your profile menu, settings, and captions. Turn it off in Settings if you need to access those.

## Install on Chrome (or any Chromium browser: Edge, Brave, Opera, Arc)

No account or payment required.

1. Click the green **Code** button on this page, then **Download ZIP**
2. Unzip the folder anywhere on your computer
3. Open your browser and go to `chrome://extensions` (or `edge://extensions`, `brave://extensions`, etc.)
4. Turn on **Developer mode** using the toggle in the top right corner
5. Click **Load unpacked** and select the unzipped `BareTube` folder
6. Done. Open YouTube and enjoy the clean view.

## Install on Firefox

1. Download and unzip the folder (same as steps 1 and 2 above)
2. Go to `about:debugging#/runtime/this-firefox`
3. Click **Load Temporary Add-on**
4. Open the unzipped folder and select the `manifest.json` file

> Note: Firefox temporary add-ons are removed when the browser closes. For a permanent install on Firefox the extension would need to be signed. Chrome, Edge and Brave work permanently with the steps above.

## Updating

When a new version is released, just download the ZIP again, replace the old folder, and click the refresh icon on the extension card in `chrome://extensions`.
